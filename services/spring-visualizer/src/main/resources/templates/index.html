FileShare Visualizer

SOAP Service Client

This application demonstrates consuming a SOAP web service from a Spring Boot application.

API Endpoints
  • GET /api/visualizer/files/{fileId} - Get file info
  • GET /api/visualizer/users/{userId}/files - Get user's files
  • GET /api/visualizer/files/{fileId}/download - Get download URL
  • DELETE /api/visualizer/files/{fileId}?userId={userId} - Delete file
